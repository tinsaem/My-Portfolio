1. Displaying Products Without Props
First, let us display three products without using props.
function State() {
  return (
    <div className="app">
      <h1>Product List</h1>

      <div className="product-container">
        <div className="product-card">
          <h3>Wireless Earbuds</h3>
          <p>$12.99</p>
        </div>

        <div className="product-card">
          <h3>Smart Watch</h3>
          <p>$18.99</p>
        </div>

        <div className="product-card">
          <h3>Bluetooth Speaker</h3>
          <p>$22.99</p>
        </div>
      </div>
    </div>
  );
}

export default State;
This code displays all three products:
Wireless Earbuds
$12.99

Smart Watch
$18.99

Bluetooth Speaker
$22.99
This works correctly.
But notice that we are repeating the same structure three times:
<div className="product-card">
  <h3>...</h3>
  <p>...</p>
</div>
Only the product title and price are different.
The structure is the same.

2. Creating a Component Without Props
To avoid writing the same product card structure directly inside the main component, we can create a separate component.
function ProductCard() {
  return (
    <div className="product-card">
      <h3>Wireless Earbuds</h3>
      <p>$12.99</p>
    </div>
  );
}

function State() {
  return (
    <div className="app">
      <h1>Product List</h1>

      <div className="product-container">
        <ProductCard />
        <ProductCard />
        <ProductCard />
      </div>
    </div>
  );
}

export default State;
This code uses a component, but there is still a problem.
The output will be:
Wireless Earbuds
$12.99

Wireless Earbuds
$12.99

Wireless Earbuds
$12.99
The same product is displayed three times because the values inside ProductCard are fixed.
So now we need a way to send different title and price values into the component.
That is where props are useful.

3. Displaying Products With Props
Props allow us to pass data into a component.
Now let us update the ProductCard component so it can receive title and price.
function ProductCard({ title, price }) {
  return (
    <div className="product-card">
      <h3>{title}</h3>
      <p>${price}</p>
    </div>
  );
}
Here, title and price are props.
They are received by the component here:
function ProductCard({ title, price }) {
They are displayed here:
<h3>{title}</h3>
<p>${price}</p>
Now we can pass different values into the same component.
function ProductCard({ title, price }) {
  return (
    <div className="product-card">
      <h3>{title}</h3>
      <p>${price}</p>
    </div>
  );
}

function State() {
  return (
    <div className="app">
      <h1>Product List</h1>

      <div className="product-container">
        <ProductCard title="Wireless Earbuds" price={12.99} />
        <ProductCard title="Smart Watch" price={18.99} />
        <ProductCard title="Bluetooth Speaker" price={22.99} />
      </div>
    </div>
  );
}

export default State;
Now the output is:
Wireless Earbuds
$12.99

Smart Watch
$18.99

Bluetooth Speaker
$22.99
This time we are using one reusable component, but each product displays different data.

4. What Are Props?
Props are data passed into a component.
Example:
<ProductCard title="Smart Watch" price={18.99} />
Here, we pass two props:
title = "Smart Watch"
price = 18.99
The ProductCard component receives those props:
function ProductCard({ title, price }) {
Then it displays them:
<h3>{title}</h3>
<p>${price}</p>
So the simple meaning is:
Props = data passed into a component

5. Advantage of Props
Without props, we repeated the product card structure many times:
<div className="product-card">
  <h3>Wireless Earbuds</h3>
  <p>$12.99</p>
</div>

<div className="product-card">
  <h3>Smart Watch</h3>
  <p>$18.99</p>
</div>

<div className="product-card">
  <h3>Bluetooth Speaker</h3>
  <p>$22.99</p>
</div>
With props, we create one reusable component:
function ProductCard({ title, price }) {
  return (
    <div className="product-card">
      <h3>{title}</h3>
      <p>${price}</p>
    </div>
  );
}
Then we pass different data into it:
<ProductCard title="Wireless Earbuds" price={12.99} />
<ProductCard title="Smart Watch" price={18.99} />
<ProductCard title="Bluetooth Speaker" price={22.99} />
The advantage of props is:
Props make components reusable.
Props reduce repeated code.
Props allow one component to display different data.

6. Using Props With an Array
Now let us store the products in an array.
const products = [
  {
    title: "Wireless Earbuds",
    price: 12.99,
  },
  {
    title: "Smart Watch",
    price: 18.99,
  },
  {
    title: "Bluetooth Speaker",
    price: 22.99,
  },
];
Instead of writing each ProductCard manually, we can use .map().
{products.map(function (item) {
  return (
    <ProductCard
      key={item.title}
      title={item.title}
      price={item.price}
    />
  );
})}
Here, .map() loops through the products array.
For each product, it creates a ProductCard.
This part passes props:
title={item.title}
price={item.price}
So each product object sends its own title and price into the component.
Complete example:
function ProductCard({ title, price }) {
  return (
    <div className="product-card">
      <h3>{title}</h3>
      <p>${price}</p>
    </div>
  );
}

function State() {
  const products = [
    {
      title: "Wireless Earbuds",
      price: 12.99,
    },
    {
      title: "Smart Watch",
      price: 18.99,
    },
    {
      title: "Bluetooth Speaker",
      price: 22.99,
    },
  ];

  return (
    <div className="app">
      <h1>Product List</h1>

      <div className="product-container">
        {products.map(function (item) {
          return (
            <ProductCard
              key={item.title}
              title={item.title}
              price={item.price}
            />
          );
        })}
      </div>
    </div>
  );
}

export default State;
At this point, we are using props to pass product data into the ProductCard component.
